[Ilyas Agadilov](https://github.com/l1lgadil1).

## About project:

Интернет-магазин, с возможностью добавлять товар в корзину, удалять из корзины, изменять количество товаров в корзине.

Приложение имеет следующие страницы: <br>
• На главную (/)<br>
• Корзина (/cart)

### Стек технологий:

• ReactJs <br>
• DummyJSON <br>
• TailwindCSS <br>
• ReduxToolKit <br>
• Redux-persist <br>
• React-router-dom <br>
